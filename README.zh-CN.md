# 星伴 AI

[简体中文](./README.zh-CN.md) · [English](./README.en.md) · [文档索引](./docs/README.md)

星伴 AI 是面向个人电脑的本地优先 AI 工作台，将聊天、智能体、项目资源管理、代码编辑、终端与 ComfyUI 图片生成整合在一个桌面应用中。

## 核心能力

### 本地模型与聊天

- 自动发现并连接 LM Studio、Ollama 与兼容 OpenAI 接口的本地模型服务。
- 支持流式正文、实时思考过程、Markdown、代码块、表格、公式、链接与图片。
- 支持 Chat、Agent、图片三种会话模式，并按会话持久化当前模式。
- 支持附件、截图、图片大图预览、`@文件` 与编辑器选区引用。
- 长消息自动折叠，长会话使用虚拟列表与前端输出队列保持流畅。
- 会话支持搜索、快速导航、本地缓存与输入输出 Token 统计。

### Agent 与工具

- 支持文件检索、区间读取、精确文本替换、行区间替换、插入、创建、删除、移动与命令执行。
- 位置未知时优先使用 `search_files` 定位关键词和行号，再按区间读取上下文。
- 用户选区只作为初始锚点；读取选区上下文后，Agent 仍可检索其他文件、会话历史或网页资料。
- 编辑现有文件前强制读取最新内容，并通过文件指纹与用户编辑锁阻止覆盖用户改动。
- `replace_in_file` 匹配失败后强制重新读取；重复失败参数会被拦截并切换为行区间替换。
- 工具模块展示真实函数名、目标文件、操作范围、耗时、结果与错误原因。
- 支持只读、修改需确认、自动修改和全自动权限模式。

### 编辑器与项目

- 多项目工作区、当前工作目录切换与树形资源管理器。
- 文件和目录支持新建、重命名、删除、复制路径、系统打开与资源管理器定位。
- 多标签编辑、标签拖拽排序、独立搜索、Markdown 预览、公式渲染与行级 Diff。
- 文件被 Agent、终端或外部程序修改后自动刷新。
- 支持编辑器主题选择、缩略图、总字数与实时保存。
- 集成基于 xterm 的持续交互终端。

### 网页与图片

- Chat 与 Agent 均可自主调用网页搜索和网页正文读取。
- 支持指定网站检索、多入口并发查询、代理访问、来源记录与失败原因展示。
- ComfyUI 工作流自动发现，支持工作流选择、模型、Steps、尺寸或比例参数。
- 图片任务使用独立队列，支持停止生成、耗时展示、大图预览、另存为和打开文件夹。

### 上下文与统计

- 读取本地模型返回的真实上下文长度与 Token 用量。
- 达到上下文窗口 90% 时调用当前本地模型压缩旧历史，并保留关键约束、文件状态和任务信息。
- 实时记录输入 Token、输出 Token、调用次数与每日月历统计。
- 当前请求默认只携带本轮目标；较早会话记录按需检索，避免重复处理历史任务。

## 环境要求

- Windows 10 或 Windows 11
- Node.js 22 或更高版本（建议）
- npm
- 至少一种本地模型服务：LM Studio、Ollama 或兼容 OpenAI API 的服务
- 图片生成功能需要可访问的 ComfyUI 服务

## 开发启动

```powershell
npm install
npm run dev
```

## 检查与构建

```powershell
npm run typecheck
npm run build
```

## 生成 Windows 安装包

```powershell
npm run dist
```

安装包输出到 `release` 目录。

## 常用脚本

| 命令 | 说明 |
| --- | --- |
| `npm run dev` | 启动 Electron 开发环境 |
| `npm run typecheck` | 执行 TypeScript 类型检查 |
| `npm run build` | 构建主进程、预加载脚本与渲染界面 |
| `npm run pack` | 生成未安装的应用目录 |
| `npm run dist` | 生成 Windows NSIS 安装包 |

## 项目结构

```text
src/main/       Electron 主进程、Agent、模型、文件与 ComfyUI 服务
src/preload/    安全的渲染进程桥接接口
src/renderer/   React 界面、状态管理、编辑器与聊天组件
src/shared/     主进程与界面共享类型
scripts/        图标构建与性能检查脚本
docs/           产品与开发文档
build/          应用图标资源
```

## 本地数据与隐私

- 会话、设置和界面状态保存在本机应用数据目录。
- 工作区文件只在用户选择的项目范围内读取或修改。
- 命令与写入操作受当前权限模式控制。
- 网页搜索仅在用户明确要求、任务依赖最新信息或现有资料不足时调用。
- `.gitignore` 已排除依赖、构建产物、安装包和本地缓存。

## 当前状态

项目仍在快速迭代，建议在重要文件上使用 Git，并在执行全自动 Agent 任务前确认当前工作区状态。

# 星伴 AI｜功能规划

> 定位：面向个人用户的本地 AI 伙伴，不做账号、团队、云同步与商业后台，重点就是聊天顺手、改文件方便、创作够快，老铁自己用着爽才是硬道理！

## 1．界面结构

```text
┌──────┬──────────────────────────┬────────────────────────────────────┐
│功能栏│ 左侧智能面板             │ 右侧文件工作区                     │
│      │                          │                                    │
│聊天  │ 会话／智能体／MCP        │ 文件树／搜索／改动                 │
│智能体│ 消息与工具调用过程       │ 多标签编辑器                       │
│搜索  │ 任务清单                 │ 预览／差异对比                     │
│MCP   │ 输入框                   │                                    │
│提示词│                          │                                    │
│设置  │                          │                                    │
├──────┴──────────────────────────┴────────────────────────────────────┤
│ 状态栏：模型｜上下文｜工作区｜智能体状态｜MCP 状态                   │
└──────────────────────────────────────────────────────────────────────┘
```

### 功能栏

- **聊天**：普通问答与会话管理，打开就聊，主打一个丝滑！
- **智能体**：读取文件、修改文件、执行命令、调用 MCP，真正开始干活！
- **搜索**：搜索文件名、文件内容与会话内容，找东西不抓瞎！
- **MCP**：管理服务、工具、资源与提示模板，整活入口安排上！
- **提示词**：管理系统提示、快捷指令与智能体预设，越用越顺手！
- **设置**：管理模型、界面、快捷键、存储与工具权限，一处搞定！

### 左侧智能面板

- 宽度可拖动，支持折叠与快捷键聚焦！
- 顶部显示当前会话、模型、模式与上下文用量！
- 中部展示消息、推理状态、工具调用、任务清单与错误信息！
- 底部提供多行输入、附件、文件引用、发送、停止与清空按钮！
- 智能体运行时仍可输入补充要求，直接纠偏，别让模型闷头瞎冲哈哈哈！

### 右侧文件工作区

- 文件树、全文搜索、改动列表使用标签切换！
- 文件支持多标签打开、固定、关闭其他与恢复已关闭标签！
- 编辑器支持语法高亮、行号、缩略图、自动换行与查找替换！
- 支持 Markdown 预览、常见图片预览、纯文本与代码编辑！
- 支持左右差异视图，并可逐段接受或拒绝智能体修改！
- 底部可展开终端与输出面板，不需要时直接收起，干净利索！

## 2．聊天功能

### 会话管理

- 新建、重命名、删除、固定与搜索会话！
- 会话自动保存，并记住使用的模型与工作区！
- 支持会话分支，从任意消息重新提问并保留旧内容！
- 支持编辑自己的历史消息后重新生成！
- 支持导出为 Markdown 或纯文本！
- 支持清空当前上下文，但保留会话记录！

### 消息能力

- 流式输出、停止生成、继续生成与重新生成！
- Markdown、代码块、表格、公式与流程图渲染！
- 代码块支持复制、插入文件、应用到文件与新建文件！
- 长回复支持折叠、目录跳转与消息内搜索！
- 工具调用以卡片展示，可展开查看参数和结果！
- 错误消息提供重试按钮，不让用户对着报错干瞪眼哈哈哈！

### 输入能力

- 支持拖入文件、文件夹、图片与文本片段！
- 使用 `@文件名` 引用工作区文件！
- 使用 `@文件夹名` 引用目录结构或选定范围！
- 使用 `/指令` 调用预设提示词！
- 支持粘贴截图，视觉模型可直接识别！
- 支持输入历史与常用问题收藏！

## 3．模型功能

### 支持的本地模型服务

- Ollama！
- LM Studio！
- llama.cpp 服务端！
- 兼容 OpenAI 接口格式的自定义地址！

### 模型管理

- 自动发现本机 Ollama 模型！
- 手动添加、编辑、复制与删除连接！
- 测试连接并显示可用模型列表！
- 每个模型单独设置上下文长度、温度、最大输出与系统提示！
- 标记工具调用、图片识别与结构化输出能力！
- 设置全局默认模型，也可给不同智能体指定不同模型！
- 会话中随时切换模型，并选择是否保留已有上下文！
- 保存常用参数预设，例如“快速问答”“写代码”“长文分析”！

### 上下文管理

- 状态栏展示当前上下文占用比例！
- 文件内容按需加入，不把整个项目一次塞进去！
- 工具返回过长时自动截取，并保留完整结果供用户展开！
- 长会话自动生成摘要，同时保留近期消息！
- 用户可查看当前发送给模型的上下文组成，透明又好用！

## 4．智能体功能

### 三种工作模式

| 模式 | 功能 |
|---|---|
| 问答 | 只分析和回答，不修改文件 |
| 编辑 | 可读取与修改工作区文件，不执行系统命令 |
| 自动 | 可修改文件、执行命令并调用 MCP 工具 |

### 任务执行

- 接收目标后生成简短任务清单！
- 自动判断需要读取的文件与工具！
- 每完成一步就更新状态，并显示当前动作！
- 支持暂停、继续、停止与重新执行失败步骤！
- 用户可在运行途中补充要求或修改目标！
- 任务结束后总结完成内容、改动文件与失败项目！

### 内置文件工具

- 列出目录与文件！
- 读取整个文件或指定行段！
- 按文件名搜索！
- 按内容全文搜索！
- 创建文件与文件夹！
- 修改、移动、重命名与删除文件！
- 批量应用补丁！
- 查看文件差异！
- 恢复单个文件或整次任务的改动！

### 内置命令工具

- 在当前工作区执行命令！
- 设置工作目录与超时时间！
- 实时展示命令输出！
- 支持停止正在运行的命令！
- 记住常用命令并支持一键重跑！
- 删除文件、安装依赖与工作区外操作可选择确认，其余动作允许自动执行！

### 版本控制辅助

- 查看仓库状态与文件差异！
- 展示当前分支与最近提交！
- 智能体可读取差异并总结改动！
- 支持暂存选中文件！
- 支持生成提交说明！
- 提交与推送保留手动按钮，个人用也别一激动把半成品送上去哈哈哈！

### 检查点

- 每次智能体任务开始前创建文件检查点！
- 修改后可查看本次全部差异！
- 支持恢复单个文件！
- 支持撤销本次任务全部文件修改！
- 命令产生的外部改动只记录日志，不承诺全部恢复，省得假装会时光倒流哈哈哈！

## 5．MCP 功能

### 服务连接

- 支持本地标准输入输出服务！
- 支持 Streamable HTTP 服务！
- 支持通过表单添加服务！
- 支持粘贴常见配置直接导入！
- 支持启动、停止、重启与自动连接！
- 显示运行状态、错误信息与实时日志！

### MCP 内容

- 展示每个服务提供的工具！
- 展示资源并支持读取与加入聊天！
- 展示提示模板并支持一键填入输入框！
- 支持工具、资源与提示模板搜索！
- 可单独启用或禁用某个服务！
- 可单独启用或禁用某个工具！
- 工具调用卡片展示参数、耗时、结果与错误！
- 支持 MCP 服务需要的环境变量，并在设置界面编辑！

### 调用方式

- 普通聊天可手动选择工具！
- 智能体模式可自动选择工具！
- 工具可设置“每次确认”“本次会话允许”“始终允许”！
- 常用工具可加入快捷操作区！
- 工具过多时只向模型提供相关工具，避免上下文被工具说明塞满！

## 6．文件工作区功能

### 工作区

- 打开单个文件夹作为工作区！
- 最近工作区列表！
- 重新加载与关闭工作区！
- 记住每个工作区的打开文件、会话与模型！
- 支持忽略目录，例如依赖目录、构建目录与版本控制内部目录！
- 工作区可放置自定义说明文件，用于告诉智能体项目规则！

### 文件树

- 新建、重命名、移动、复制与删除！
- 展开、折叠与刷新！
- 显示文件修改状态！
- 右键发送文件或目录到聊天！
- 右键让智能体解释、修改、重构或生成测试！
- 支持拖动文件调整位置！

### 搜索

- 文件名模糊搜索！
- 全文搜索！
- 大小写、完整单词与正则表达式开关！
- 按文件类型与目录筛选！
- 搜索结果点击后跳转到具体行！
- 支持跨文件替换，执行前展示改动预览！

### 编辑器

- 自动保存与手动保存切换！
- 撤销、重做、格式化与快捷注释！
- 多光标、括号匹配与缩进辅助！
- 常见语言语法高亮！
- 跳转到指定行！
- 文件编码与换行格式显示！
- 外部修改后提示重新加载或对比！

### 预览

- Markdown 双栏预览！
- 常见图片查看与缩放！
- PDF 使用系统默认程序打开！
- 其他文件可选择纯文本查看或外部打开！

## 7．搜索与项目理解

### 基础项目理解

- 首次打开工作区时扫描目录与文件类型！
- 自动识别项目说明、依赖配置与常用入口文件！
- 提问时优先检索相关文件，不直接读取整个项目！
- 智能体可生成项目概览、目录说明与运行方式！

### 可选语义检索

- 使用本地嵌入模型建立工作区索引！
- 支持自然语言搜索代码与文档！
- 可手动重建、暂停或删除索引！
- 显示索引进度与占用空间！
- 首版先做文件名与全文搜索，语义检索后补，别上来就把自己卷寄了哈哈哈！

## 9．终端与输出面板

- 右侧底部提供终端标签！

## 11．错误与状态提示

- 模型未启动时显示连接按钮与排查提示！
- 模型请求失败时显示错误原因与重试按钮！

- 工具执行超时后允许重试或停止！
- 文件冲突时打开差异视图，不静默覆盖！
- 上下文接近上限时提前提醒并提供摘要按钮！
- 智能体停止后保留已完成结果，可从失败步骤继续！

## 14．推荐技术方案

| 模块 | 方案 |
|---|---|
| 桌面应用 | Electron |
| 界面 | React＋TypeScript |
| 文件编辑器 | Monaco Editor |
| 样式 | Tailwind CSS |
| 状态管理 | Zustand |
| 本地存储 | SQLite |
| 模型连接 | Ollama 与 OpenAI 接口格式适配器 |
| MCP | 官方 TypeScript SDK |
| 终端 | xterm.js＋node-pty |

技术结构保持简单：

```text
桌面主进程
  ├─ 文件操作
  ├─ 命令执行
  ├─ 模型连接
  ├─ MCP 连接
  └─ 本地数据

界面进程
  ├─ 聊天与智能体
  ├─ 文件树与编辑器
  ├─ 终端与日志
  └─ 设置页面
```

## 15．开发顺序

### 第一阶段：主界面与文件区

- 接入 Ollama 与自定义接口地址！

### 第四阶段：命令与终端

- 完成终端面板、命令执行、实时输出与停止！

### 第五阶段：MCP

- 完成服务配置、工具发现、调用卡片与日志！

### 第六阶段：功能补齐
